# Visual Studio 恢复默认设吹

## 一、通过 " 导入和导出设置 " 实现 VS 恢复默认设置

　　工具 → 导入和导出设置 → 重置所有设置 → 下一步 → 选择 " 是否保存当前设置 "，下一步 → 选择 " 要重置的开发语言 " → 完成。
