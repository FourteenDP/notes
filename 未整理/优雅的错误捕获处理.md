# 优雅的错误捕获处理

```javascript
async function errorCaptured(asyncFunc){
  try{
    let res = await asyncFunc()
    return [null, res]
  }catch(e){
    return [e, null]
  }
}
```

## 使用

```javascript
let [err, res] = await errorCaptured(asyncFunc)
```
