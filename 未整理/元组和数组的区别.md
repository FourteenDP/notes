# 元组和数组的区别

## 元组

元组是一个固定长度的数组，它的长度是在定义时确定的。

```ts
let x: [string, number];
x = ["hello", 10]; // OK
x = [10, "hello"]; // Error
```

## 数组

数组是一个可变长度的数组，它的长度是在运行时确定的。

```ts
let list: number[] = [1, 2, 3];
let list: Array<number> = [1, 2, 3];
```
