# TypeScript

TypeScript 是由微软开发的开源编程语言。它是 JavaScript 的一个超集，主要提供了类型系统和对 ES6 的支持。

**超集**是指一个集合包含另一个集合的所有元素，即 A 包含 B，那么 A 就是 B 的超集。

简单的来说，TypeScript 是在 JavaScript 的基础上增加了类型系统以及其它特性，使得 JavaScript 更加强大。
